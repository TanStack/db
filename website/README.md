# TanStack DB Website

Official website for TanStack DB - A client-side database with colocated reactive queries, effortless optimistic mutations, and query-driven sync.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- Modern, clean design inspired by graphql.org
- Dedicated Query-Driven Sync feature page
- Comprehensive documentation links
- Community resources and contribution guidelines
- Responsive design for all devices

## Built With

- React 18
- Vite
- React Router
- TypeScript
